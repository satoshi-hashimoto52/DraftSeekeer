# frontend/src/utils/color.ts

## 要約
- HSL/HEX の変換ユーティリティ。
- 不正値は黒 (#000000) にフォールバック。

## 目的/責務
- colorMap の正規化

## 公開API
- `isHexColor`
- `hslToHex`
- `normalizeToHex`

## 入出力データ
- 入力: 文字列
- 出力: HEX 色

## 依存
- なし

## 重要アルゴリズム/落とし穴
- HSL 文字列の正規表現にマッチしない場合は黒

## テスト観点
- 正常な #RRGGBB
- 正常な hsl()
- 不正な文字列
- 空文字
- 境界値 (0, 360)

## 変更時の注意
- 色表現の変更は UI に影響
